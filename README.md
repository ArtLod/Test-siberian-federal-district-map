# Карта Сибирского ФО 

Учебный проект по созданию интерактивной карты с использованием библиотеки Leaflet.

## Запуск

Открыть `index.html` в любом браузере. 

## Данные

Данные для показателей взяты из статистического сборника
«Регионы России» за 2024.
